未付费用户展示内容：

1. 以产品功能宣传为主，提高成单效率，缩短付费流程，并增加优惠购买活动

2. 开放免费试用功能，试用期间不做接管，主要以园所形象打造为突破点，开放大量海报使用功能，塑造专业性并暗含付费引导

    同时收集用户信息（幼儿园信息、手机号）便于后期电话回访促单


已付费用户展示内容：

1.  将近期新功能微信公众号接管流程与付费流程紧密结合并风格统一化

2.  将流量池 早安海报 + 行事历通知功能优化升级，开放海报文案自定义、二维码自定义，图文链接 / 海报 / 模板消息通知形式任选功能

3.  增加宣传大使数据统计功能，提高用户海报自转发，自宣传能力

4.  增加日常重要事项 及 重要表格等内容使用通知
